آموزش و سوالات متداول

آموزش وارد کردن کانفیگ در NPV Tunnel

گام ۱: اپ NPV Tunnel را باز کنید.

گام ۲: روی علامت + (افزودن) در پایین صفحه بزنید.

گام ۳: گزینه Import config from clipboard یا from file را انتخاب کنید.

گام ۴: کانفیگ کپی‌شده را Paste کنید یا فایل را انتخاب نمایید.

گام ۵: کانفیگ اضافه شد → آن را انتخاب کنید و دکمه اتصال را بزنید.

نکته: اگر اتصال برقرار نشد، SNI را چک کنید یا کانفیگ جدید بگیرید.
